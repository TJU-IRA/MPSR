# MPSR
MPSR is a Multi-scale Perception Network for Infrared Image Super-Resolution that has been submitted to TIP. The code will be made public after the acceptance of the paper.
## 1.Requirements
1.Python 3.10  

2.Opencv  

3.Pytorch 1.13.0  

4.torchvision  

5.imageio  

6.skimage
